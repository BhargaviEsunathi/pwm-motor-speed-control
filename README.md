# PWM Motor Speed Control

## Description
This project controls motor speed by varying the PWM duty cycle
generated using a microcontroller.

## Microcontroller / Board
Arduino UNO / ESP32 / STM32

## Programming Language
Embedded C

## Tools Used
Arduino IDE

## Key Features
- PWM signal generation
- Duty cycle variation
- Motor speed control

## What I Learned
- PWM working principles
- Timer usage in microcontrollers
- Practical motor control basics
